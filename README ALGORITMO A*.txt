README
- Qué hiciste:
  Creé un código utilizando pygames para la interfaz. Este código genera un tablero basado en las entradas del usuario. El tablero permite insertar obstáculos y marcar puntos de entrada y salida. Luego, encuentra el camino óptimo entre la entrada y la salida.
- Qué algoritmo usaste:
  Después de investigar, llegué a la conclusión de que el algoritmo A* es uno de los más utilizados, por lo que lo implementé en mi código.
- Qué aprendiste:
  Aprendí a usar el framework pygames para diseñar mi interfaz y a implementar el algoritmo A*.
